## 🔒 安全架构

- **密码哈希**：PBKDF2 (100,000次迭代 + 随机32字节Salt)
- **数据加密**：AES-256-GCM，每个字段独立IV
- **存储格式**：`ENC:<iv_base64>:<ciphertext_base64>`
- **会话管理**：sessionStorage，8小时超时，关闭浏览器自动登出
- **错误锁定**：连续输错密码触发临时锁定
- **伪装保护**：login.html默认显示Nginx 404页面，通过隐形触发器(右下角点5次)或 `?auth=1` 参数访问

## 🐛 故障排除

### 网页打开后只显示 "404 Not Found"？
这是**正常的伪装保护**。有以下两种方式进入真实登录界面：
1. 直接访问 `你的网址/login.html?auth=1`（推荐书签此URL）
2. 在页面**右下角区域快速点击5次**（无视觉提示的隐形触发器）
3. 键盘快捷键：**Ctrl + Shift + L**

### 首次使用流程：
1. 访问 `你的网址`（会自动跳转到设置密码页面）
2. 设置一个8位以上的密码
3. 登录后即可使用全部功能
